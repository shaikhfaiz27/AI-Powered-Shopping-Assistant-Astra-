# AI-Powered-Shopping-Assistant-Astra-
Built Astra, an end to end AI powered multilingual shopping assistant using Python, Streamlit, FAISS, Sentence Transformers, Gemini AI, and Pandas. Developed semantic product search with vector embeddings, ML based product ranking, budget aware recommendations, detailed product insights, review analysis, voice search, multilingual translation, and a custom PC build recommendation engine with compatibility checks. Integrated structured product, sales, specification, and review datasets to deliver personalized, intelligent shopping recommendations.
